**Курсова робота з предмету "Системне програмне забезпечення"**






Розробка утиліти для резервного копіювання файлів та даних. Розробка програми, що дозволяє автоматично створювати резервні копії важливих файлів та даних на комп'ютері з операційною системою Windows.

    1. Можливість сканування всього диска або вибраних папок для пошуку втрачених файлів.
    
    2. Підтримка різних типів носіїв, включаючи жорсткі диски, флеш- накопичувачі та карти пам'яті.
    
    3. Виявлення втрачених або видалених файлів на основі сканування диска.
    
    4. Фільтрація результатів за розширенням файлів або іншими параметрами.
    
    5. Можливість відновлення втрачених файлів з наданими результатами сканування.
    
    6. Забезпечення можливості вибору місця для збереження відновлених файлів.
    
    7. Підтримка попереднього перегляду знайдених файлів перед їх відновленням.
    
    8. Можливість перегляду текстових, графічних та відеофайлів для підтвердження їх цілісності.
